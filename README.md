## 📜 &nbsp; How I work

```diff
+ Build the hard version — short-cuts compound into rewrites
+ Local-first by default — networked features are opt-in, metered, revocable
+ Modularity over convenience — narrow surfaces, no silent coupling
+ Evidence over assumption — if docs and code disagree, code wins
+ Boring infra is good infra — IPC you can grep, permissions you can audit
- "Trust me" — every claim earns its place by being verifiable
- "We'll fix it later" — later doesn't come; do it once, do it right
```
